# 🧠 AI 黑话翻译器

> 把 AI 圈的黑话，从"听不懂的咒语"翻译成"能笑出声的常识"。
> **硬核考据，冷面吐槽，一个词读懂五层。**

## 这是什么？

一个把 AI 术语逐个"解剖"的开源项目。每个词条都按**五层结构**展开，你想读到哪一层，就停在哪一层：

| 层级 | 内容 | 适合谁 |
|------|------|--------|
| **L1 一句话梗** | 让你笑、让你转发 | 所有人 |
| **L2 生活类比** | 用火锅、职场、武侠讲直觉 | 小白 / 跨界者 |
| **L3 正经定义** | 教科书级准确解释 | 想搞清楚的人 |
| **L4 原理深挖** | 公式、论文、最小可运行 demo | 工程师 / 学生 |
| **L5 八卦与坑** | 历史渊源、提出者轶事、面试怎么考 | 看戏群众 / 求职者 |

**为什么是五层？** 因为大多数科普要么太浅（纯段子，工程师看不上），要么太硬（论文翻译，小白看不懂）。我们把两种动力焊在一起——小白为 L1 的梗转发，工程师为 L4 的深度收藏。

## 读法建议

- **只想跟人吹牛** → 读 L1 + L2，3 分钟
- **要写进简历** → 通读 L1~L4，30 分钟
- **准备面试** → 重点看 L4 + L5

## 为什么开源？

1. **学习**：写一遍比读十遍记得牢。这是作者自己的费曼学习法闭环。
2. **协作**：欢迎你投稿词条——写出来就是你的，署名归你，被转发时你也跟着涨粉。
3. **抗遗忘**：AI 圈黑话迭代太快，这个仓库当个长期的"术语考古队"。
